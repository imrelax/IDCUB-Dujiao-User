# Dujiao-Next User Web · 二次开发版本

这是一个基于 AI 辅助开发的 Dujiao-Next 用户端二次开发版本。所有功能实现、代码简洁性和运行效率均取决于 AI 模型的能力，而非开发者个人。如果您对此有所顾虑，请谨慎使用。

由于官方版本缺少许多功能，本项目采用了自定义配置文件来实现这些特性。我会持续跟进官方更新进度，确保与上游保持同步。

- **官方尚未实现的功能**：我会在本版本中继续维护和扩展。
- **官方后续实现的功能**：我会基于官方实现进行适配，减少对配置文件的依赖，使操作更加简便。

## 🔧 配置文件说明

本项目的主要自定义功能均通过 `src/assets/xiabibi.json` 配置文件实现，包括：

- 首页幻灯片/横幅配置
- 教程中心对接配置
- 热销标签配置
- 公告弹窗配置

请在使用前仔细编辑该配置文件，替换示例数据为您的实际业务内容。
