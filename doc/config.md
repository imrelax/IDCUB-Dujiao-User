## 配置文件说明

本项目的自定义功能主要通过以下配置文件实现，请仔细阅读并正确配置。

### vite.config.ts

此文件为 Dujiao 主题的构建配置文件。为了适配 Vercel 部署环境，我修改了开发服务器的代理目标地址。

**配置说明**：
- **Vercel 部署**：请将代理目标地址修改为您的实际 API 地址
- **本地开发,编译部署等**：请将代理目标地址修改为 `target: 'http://localhost:8080'`

### vercel.json

此文件是 Vercel 平台的反向代理配置文件，用于在生产环境中正确路由 API 请求。

**重要提示**：
- 此文件为 Vercel 部署必需文件
- 请务必将其中的 API 地址修改为您实际的API接口地址

### src/assets/xiabibi.json

此文件是本项目的核心配置文件，所有自定义功能均在此文件中进行配置。

**功能涵盖**：
- 首页幻灯片/横幅配置
- 教程中心对接配置
- 商品热销标签配置
- 公告弹窗配置
- 友情链接配置
- 其他页面优化配置

**使用说明**：
配置文件中已包含详细的注释说明，请根据实际业务需求参考注释进行相应设置。
