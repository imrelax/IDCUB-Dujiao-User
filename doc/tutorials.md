## 教程中心功能

教程中心是本项目新开发的功能模块，用于展示教程文章和知识库内容。

### 功能背景

Dujiao 官方原本提供博客功能，但由于暂时不支持文章分类管理，我们采用了 WordPress 对接方案来实现教程中心功能。这是一个有效的临时解决方案，未来如果官方支持文章分类，我们可能会回归使用原生博客功能。

### 功能实现原理

教程中心通过调用 WordPress REST API 接口获取内容，实现动态教程展示。

**配置方式**：
1. 在配置文件中设置 WordPress 站点的接口地址
2. 指定需要显示的一级分类 ID
3. 系统自动获取对应的二级分类及文章信息

### 功能特性

1. **分类展示**
   - 自动显示指定一级分类下的所有二级分类
   - 按分类组织教程文章，便于用户浏览

2. **文章浏览**
   - 点击文章标题可直接进入文章详情页
   - 提供完整的文章阅读体验

3. **动态内容**
   - 所有教程内容从 WordPress 站点实时获取
   - 内容更新即时同步到前端展示

### 配置说明

教程中心的所有配置均在 `src/assets/xiabibi.json` 文件中进行：
- `apiBase`: WordPress 站点接口地址
- `tutorialParentCategoryId`: 教程父级分类 ID
- `categorySettings`: 分类显示设置

请根据实际需求在配置文件中进行相应设置。
