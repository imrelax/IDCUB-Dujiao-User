## 商品详情页优化

针对商品详情页，我们进行了多项优化，提升了页面的视觉效果和用户体验。

### 商品图片与信息区域

左侧商品图片区域与右侧文本区域实现了等高设计，使页面整体视觉效果更加协调统一。

**主要优化细节**：

1. **商品热卖标签**
   - 在商品图片右上角添加了可配置的热卖标签
   - 支持通过配置文件自定义标签内容、颜色样式
   - 可指定需要显示标签的具体商品 ID

2. **商品 ID 显示**
   - 在右侧信息区域增加了商品 ID 显示（如：ID 1234）
   - 方便客户沟通时快速定位具体商品
   - 便于运营人员管理商品信息

3. **促销标签功能**
   - 在商品价格区域增加了可配置的促销标签
   - 支持自定义标签文本内容
   - 可指定需要显示促销标签的商品编号
   - 提供灵活的促销信息展示方式

### 商品详情内容区域

此部分改动相对较小，主要优化了布局结构，提升了内容展示效果。

**布局优化**：
- 将页面分为左右两侧布局
- 左侧为商品导航菜单
- 右侧为商品详细描述内容

**自适应设计**：
- 左侧边栏高度会自动跟随右侧详情内容高度调整
- 当右侧详情内容较少时，左侧边栏保持初始高度
- 确保页面布局的平衡和美观

**数据来源**：
- 此部分内容完全从网站后台获取
- 无需通过配置文件进行自定义设置
- 保持与官方数据源的一致性

### 配置说明

商品详情页的自定义功能主要通过 `src/assets/xiabibi.json` 配置文件实现：
- 热卖标签配置：设置标签文本、颜色、适用商品
- 促销标签配置：设置促销信息、适用商品范围
- 其他视觉优化配置

请根据实际业务需求在配置文件中进行相应设置。
