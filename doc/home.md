## 首页配置优化

### 🪟 首页弹窗

进入首页后，系统会自动弹出公告弹窗。弹窗内容来源于后台公告系统，可实现重要信息的即时推送。

**配置步骤**：
1. 在后台管理系统中发布一篇公告
2. 在配置文件中设置该公告的 ID
3. 弹窗将自动显示对应公告的内容

### 🎨 首页幻灯片横幅

首页幻灯片横幅完全通过配置文件进行定制，无需后台操作，提供高度灵活的展示方式。

**可配置项**：
- 文案内容：自定义标题和描述文本
- 背景颜色：支持渐变背景色设置
- 右侧图标：可自定义 SVG 图标内容
- 跳转链接：设置点击后的目标页面

### 📢 首页公告栏

在幻灯片下方新增公告栏区域，用户点击可直接跳转到公告详情页面。

**配置方式**：
- 在配置文件中设置需要显示的公告 ID
- 页面将自动显示对应公告的标题
- 点击公告标题将进入该公告的详细内容页面

### 🏷️ 商品热销标签

在商品列表中为特定商品添加可配置的热销标签，提升商品曝光度。

**配置特性**：
- 标签文本：支持自定义显示文字
- 颜色样式：可自定义标签背景和文字颜色
- 权重排序：支持权重参数，按权重高低排序显示

### 📦 库存显示控制

新增智能库存判断逻辑，优化商品展示策略，提升用户体验。

**功能说明**：
- 当商品库存为 0 时，全站自动隐藏该商品
- 补充库存后，商品自动恢复显示
- 避免用户看到无法购买的商品

**设计背景**：
由于许多商品是从上游供应商导入，供应商经常出现部分商品售罄的情况。如果继续显示无库存商品，会导致用户频繁遇到无法购买的情况，严重影响购物体验。此调整可确保用户看到的都是可购买的商品。
